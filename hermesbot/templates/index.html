SSH Ključ (Private modeli)
Za privatne HF GGUF modele. Dodaj Ollama SSH key na HF account.
1. Pokreni: cat ~/.ollama/id_ed25519.pub
2. Idi na huggingface.co/settings/keys
3. Dodaj SSH key
4. Sada možeš: ollama run hf.co/tvoj-user/privatni-repo
HF_SSH_KEY env varijabla (opcionalno)

Telegram Bot
Poveži Hermesa sa Telegram botom
not set
Env: TELEGRAM_BOT_TOKEN
Dobij token od @BotFather na Telegramu
Setup koraci:
1. Otvori Telegram → @BotFather
2. /newbot → unesi ime → dobij token
3. Postavi TELEGRAM_BOT_TOKEN env var na Railway
4. Redeployaj → bot je aktivan
5. Pošalji /start svom botu
Telegram Komande
/start — Pokretanje bota
/new — Nova konverzacija
/memory — Pregled memorije
/skills — Lista skillova
/model — Trenutni model
/status — Status Ollame

Discord Bot
Hermes na Discord serveru
not set
Env: DISCORD_BOT_TOKEN
discord.com/developers/applications → Bot → Token
Setup:
1. discord.com/developers/applications → New App
2. Bot → Reset Token → kopiraj
3. OAuth2 → bot scope → Send Messages, Read History
4. Pozovi bota na server
5. Postavi DISCORD_BOT_TOKEN na Railway

Railway Deploy Guide
1. Fork ovaj repo na GitHub
2. Otvori railway.app → New Project
3. Deploy from GitHub repo
4. Dodaj Ollama servis (Docker image: ollama/ollama)
5. Postavi OLLAMA_HOST na Ollama internal URL
6. Postavi ostale env vars
7. Deploy → URL je dostupan!
